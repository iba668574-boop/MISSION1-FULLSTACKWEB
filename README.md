# Mission 4 Full Stack Web

## Features

* User Registration
* User Login
* Protected Route
* Item Search

## Tech Stack

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MongoDB

## Installation

```bash
npm install
npm start
```

## API Endpoints

### Register

POST /register

### Login

POST /login

### Profile

GET /profile

### Items

GET /items
